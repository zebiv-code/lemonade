# Lemonade Stand

Web recreation of the classic 1979 Apple II business simulation game by Bob Jamison (MECC). Features authentic low-resolution graphics and speaker-driven sound effects.

**[Live Demo](https://zebiv.com/lemonade/)**

## Features

- Faithful recreation of the original game logic
- Apple II low-resolution (40×48) color graphics simulation
- Authentic sound effects recreated from original POKE values
- Weather system, pricing strategy, advertising
- Retro green phosphor text rendering

## Tech

- Vanilla HTML/CSS/JavaScript
- Canvas 2D for lo-res graphics
- Web Audio API for Apple II speaker sounds
- ES modules
- No build step, no dependencies
